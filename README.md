# Ecommerce Website

A modern responsive e-commerce website built using Next.js and TypeScript.

## Features

- Product listing
- Product details page
- Add to cart functionality
- Wishlist system
- Login / Signup UI

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Setup

npm install  
npm run dev
